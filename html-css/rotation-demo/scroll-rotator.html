Animation is paused. A negative animation-delay tied to scroll progress scrubs through the keyframes frame by frame.

Section 05 — Done

THAT'S
IT

Three things: the CSS block, the HTML block, and one line of JS. Copy, paste, ship.

scroll-rotator.js
// ── SCROLL-ROTATOR JS ────────────── // Paste once in your <script> block. // No dependencies needed. window.addEventListener('scroll', () => { const scrolled = window.pageYOffset; const maxScroll = document.body.offsetHeight - window.innerHeight; const progress = scrolled / maxScroll; document.body.style .setProperty('--sr-scroll', progress); }, { passive: true }); // ── CUSTOMISE ────────────────────── // CSS vars to tweak: // --sr-size object width // --sr-perspective depth feel // --sr-glow-color halo color
